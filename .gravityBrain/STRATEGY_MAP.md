# STRATEGY MAP (LegoStack)

## 👤 타겟 페르소나 (Persona)
1. **인디 메이커 (The Indie Maker)**: 최소 비용으로 MVP를 만들고 싶은 1인 개발자.
2. **기술 탐험가 (The Tech Explorer)**: 최신 AI 모델의 트렌드와 미학적 디자인을 중요하게 생각하는 얼리어답터.
3. **비즈니스 기획자 (The Startup Founder)**: 서비스 규모에 따른 비용 예측(Scalability)이 필요한 창업자.

## 🗺️ 사용자 여정 (User Journey Map)
### 1. 인지 (Discovery)
- **행동**: SNS나 커뮤니티에서 "가장 예쁜 AI 리소스 허브"라는 소문을 듣고 접속.
- **심리**: "오, 디자인 예쁘네. 근데 나한테 필요한 게 있을까?"

### 2. 탐색 (Exploration)
- **행동**: 카테고리 필터를 눌러보며 다양한 브릭들을 구경. 아이콘과 이름 옆의 'Killer Feature'를 확인.
- **심리**: "복잡한 가격표 대신 '질문 100개당 가격'으로 보여주니 이해가 쏙 되네."

### 3. 시뮬레이션 (Wow Point)
- **행동**: 필요한 브릭을 스택에 담고 하단 MAU 슬라이더를 조작.
- **심리**: **(안도감)** "내가 만들 서비스가 유저 1,000명일 때 월 $20면 충분하구나! 당장 시작해야지."

### 4. 전환 및 공유 (Conversion)
- **행동**: 각 도구의 'Get Started' 링크를 누르거나, 완성된 내 스택을 공유.
- **심리**: "이 사이트 진짜 유용하다. 즐겨찾기 해두고 자주 와야지."

## 🛡️ 이탈 방지책 (Retention Strategy)
- **데이터 투명성**: "신용카드 없이 시작 가능" 같은 꿀팁을 태그로 노출하여 심리적 장벽 제거.
- **주간 트렌드**: 매주 새로운 '가장 힙한 브릭'을 업데이트하여 재방문 유도.
- **커뮤니티 검증**: 실제 개발자들의 'Aesthetic Score'를 반영하여 신뢰도 확보.
