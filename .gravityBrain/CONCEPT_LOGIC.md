# CONCEPT LOGIC (LegoStack)

## 1. 데이터 표준화 모델 (Standardized Data)
사용자가 도구 간 가격을 쉽게 비교할 수 있도록 다음 로직으로 데이터를 가공함.

- **LLM 모델**: 100만 토큰당 가격(Input/Output 가중치 3:7 적용)을 기본으로 하되, UI에는 **"일반적인 질문 1,000개당 가격"**으로 환산하여 노출.
- **이미지/비디오 모델**: **"고퀄리티 생성 100건당 가격"**으로 환산.
- **인프라 모델**: DB(1GB당), Storage(10GB당), Bandwidth(100GB당)를 기준으로 **"사용자 1,000명당 예상 비용"** 환산.
- **무료 티어**: 단순히 '무료 있음'이 아니라 **"매월 공짜로 쓸 수 있는 한도(Quota)"**와 **"리셋 주기"**를 명시.

## 2. 큐레이션 및 점수 체계 (Scoring)
- **DX Score (1.0~5.0)**: 디자인 심미성, 문서화 품질, SDK 지원 여부를 종합한 '개발자 경험' 점수.
- **Killer Feature**: 단순 기능 설명이 아닌 **"이 도구가 업계 1위인 단 하나의 이유"**를 한 줄로 요약.

## 3. 유저 여정 동선 (Flow)
1. **Landing**: 네비게이션 바 로고와 깔끔한 그리드 노출.
2. **Filter**: 관심 분야(예: Vision) 필터링.
3. **Add**: 브릭 클릭 -> 하단 계산기 바에 반영.
4. **Slide**: MAU 조절 -> 실시간 총액 확인.
5. **Action**: 상세 정보 확인 및 공식 사이트 방문.
