# 체크리스트: 공유 시스템 고도화 및 번역 품질 검토

## 📋 작업 개요
- **목표**: DB 의존성을 줄인 URL 기반 상태 공유 기능을 강화하고, 공유 UI/UX를 프리미엄 수준으로 개선하며, 다국어 지원의 완성도를 높인다.
- **중점 사항**: URL Serialization, Share UI Animation, i18n Completeness.

## 🛠️ 세부 작업 항목

### 1. URL 쿼리 스트링 직렬화 (Stateless Sharing)
- [x] **Serialization Utility**: `src/lib/serialize.ts` 생성 (LZ-String 기반 객체 ↔ 문자열 변환).
- [x] **Store Sync**: 스택 상태 변경 시 URL의 `?s=` 파라미터를 실시간(혹은 디바운스)으로 업데이트하는 hook 개발.
- [x] **Hydration Logic**: 페이지 진입 시 URL에 `s` 파라미터가 있으면 DB 조회 없이 즉시 스토어에 반영.
- [x] **Test Code**: 다양한 스택 조합에 대한 직렬화/역직렬화 무결성 테스트.

### 2. 'Share' UI 및 메타 태그 최적화
- [x] **Share Button UI**: Framer Motion을 활용한 부드러운 팝업/토스트 피드백 추가.
- [x] **Social Meta Tags**: `app/[locale]/layout.tsx` 혹은 `page.tsx`에서 동적 OG 이미지 및 설명 문구 연동.
- [x] **Image Generation Refine**: `html-to-image`로 생성되는 카드 디자인을 더 프리미엄하게 조정.

### 3. 영문 번역 (i18n) 완성도 검토
- [x] **Missing Keys**: `messages/ko.json`과 `messages/en.json` 간의 키 불일치 전수 조사.
- [x] **Natural Phrasing**: 어색한 영어 표현(Konglish 등)을 자연스러운 Tech English로 교정.
- [x] **Contextual Labels**: '해석 도우미'의 비유 표현들이 영미권 문화에 맞는지 확인 (예: 강남역 → Times Square).
