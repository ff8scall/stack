# IDEATION LOG (나중에 하기로 한 기능들)

## 💤 Postponed Features (Phase 2+ 예정)

1. **사용자 로그인 및 개인 저장함**
   - **이유**: 초기 유입 단계에서는 가입 절차 없이 바로 계산기를 써보게 하는 것이 전환율에 유리함. 나중에 '저장하기' 기능이 필요할 때 도입.
   
2. **실시간 가격 크롤링 자동화**
   - **이유**: 개발 복잡도가 높음. Phase 1에서는 수동 리서치한 33종 데이터를 JSON으로 관리하며 빠른 런칭에 집중.

3. **커뮤니티 댓글 및 리뷰**
   - **이유**: 스팸 관리 및 운영 리소스 발생. 초기에는 Upvote(좋아요) 기능으로 단순화하여 운영 부담 최소화.

4. **스택 공유 이미지 생성 (OG Image Generator)**
   - **이유**: 기술적 난이도가 높으므로 Phase 1에서는 단순 링크 공유로 대체.
