# 체크리스트: Advanced SEO & Technical 고도화 (2026-04-26)

## 🎯 목표
- 단순 검색 노출을 넘어 클릭률(CTR) 개선 및 검색 엔진 신뢰도(Silo) 강화.
- 구조화 데이터 고도화 및 사용자 이동 경로(Breadcrumbs) 최적화.
- 글로벌 타겟팅 및 중복 콘텐츠 제어 완결.

## 🛠️ 작업 단계

### 1. 구조화 데이터 및 탐색 구조 (Breadcrumbs & FAQ)
- [x] 브릭 상세 페이지 및 카테고리 페이지에 **Breadcrumb (이동 경로)** UI 및 JSON-LD 추가 -> **완료**
- [x] 상세 페이지에 자동 생성형 **FAQ Schema** (예: 가격 정보, 주요 특징) 추가 -> **완료**
- [x] 실시간 데이터임을 강조하는 `Dataset` 스키마 추가 -> **완료 (상세 페이지 적용)**

### 2. 내부 링크 및 Silo 구조 강화
- [x] 상세 페이지 하단 '연관 도구' 섹션의 링크가 논리적 계층(Category -> Brick)을 따르는지 재확인 -> **Silo 구조 완결**
- [x] 메인 페이지에서 각 카테고리 실로(Silo)로의 연결성 강화 -> **Hero 하단 탐색 링크 추가 완료**

### 3. 메타데이터 CTR 최적화
- [x] Title 태그에 '2026', '최신' 등의 **파워 키워드** 및 **CTA** 포함 로직 강화 -> **적용 완료**
- [x] 동적 OG 이미지의 디자인 레이아웃이 소셜 미디어에서 정보 전달력이 높은지 재점검 -> **검증 완료**

### 4. 미디어 및 글로벌 타겟팅
- [x] 다국어 지원용 `hreflang` 설정의 국가 코드(`ko-KR`, `en-US`) 및 전역 매핑 재점검 -> **확인 완료**
- [x] 모든 이미지 포맷의 WebP/SVG 전환 여부 및 최적화 확인 -> **적용 완료**

## ⚠️ 주의사항
- **구조화 데이터 오류**: Google Rich Results Test 도구에서 오류가 없는지 데이터 구조 엄밀성 유지.
- **Breadcrumbs 일관성**: UI 상의 경로와 JSON-LD 상의 경로가 100% 일치해야 함.
